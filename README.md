# The Chemistry

An elegant and cinematic love compatibility web experience built with **HTML, CSS, and JavaScript**.

Designed with refined typography, subtle motion, luxury-inspired colors, and smooth microinteractions to create a premium romantic interface.

---

## Features

- Elegant luxury-inspired UI
- Smooth floating heart animations
- Animated compatibility percentage counter
- Dynamic relationship verdicts
- Responsive layout for desktop and mobile
- Cinematic gradients and depth effects
- Interactive hover and transition effects
- Clean typography using Google Fonts

---

## Built With

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts
  - Cinzel
  - Cormorant Garamond

## Customization

You can easily customize:

### Colors

Edit the CSS variables inside:

```css
:root {
  --crimson: #8B1A2B;
  --gold: #C4965A;
}
```

### Verdict Messages

Modify the verdict system inside:

```javascript
const verdicts = [...]
```

### Typography

Fonts can be changed through:

* Google Fonts import
* `font-family` in CSS
